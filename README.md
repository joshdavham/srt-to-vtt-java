# srt-to-vtt-java

# Test the package locally

```bash
mvn test
```

# Install the package locally

```bash
mvn clean install
```

# Import the package

```bash
  <dependency>
    <groupId>io.github.joshdavham</groupId>
    <artifactId>srt-to-vtt</artifactId>
    <version>1.0.0</version>
  </dependency>
```
